### ✅ Exportadores de Tokens (v1.3.0)

#### `scripts/export-tokens.py`
Genera tokens en múltiples formatos desde `component-manifest.json`:

```bash
# Exportar a CSS (por defecto)
python3 scripts/export-tokens.py --format css

# Exportar a JavaScript (ESM)
python3 scripts/export-tokens.py --format js

# Exportar a JSON (estructura plana)
python3 scripts/export-tokens.py --format json

# Exportar a SCSS (mixins y mapas)
python3 scripts/export-tokens.py --format scss

# Especificar salida personalizada
python3 scripts/export-tokens.py --format js --output dist/design-tokens.js
```

**Outputs disponibles:**
- `01-tokens/tokens.css` — Variables CSS custom properties
- `01-tokens/tokens.js` — Módulo ESM con `semanticTokens` y `brandTokens`
- `01-tokens/tokens.json` — Estructura JSON comprimida
- `01-tokens/tokens.scss` — Mixins SCSS con `@include brand-tokens()`

**Características:**
- ✅ Genera automáticamente desde el manifiesto (SSOT)
- ✅ Incluye valores primitivos y derivados (marca × tema)
- ✅ Comentarios con metadata (`use`, `not_for`)
- ✅ Listo para consumidores: web, apps nativas, Figma, etc.

**Integración CI:**
En `.github/workflows/validate.yml`, agregar:
```yaml
- name: Export tokens
  run: |
    python3 scripts/export-tokens.py --format css
    python3 scripts/export-tokens.py --format js
    python3 scripts/export-tokens.py --format json
```

## 🚀 Flujo de Uso

### Para desarrolladores web
```bash
# 1. Clonar kit
git clone https://github.com/promptea/ds-base-kit
cd ds-base-kit

# 2. Exportar tokens a CSS
python3 scripts/export-tokens.py --format css

# 3. Usar en tu aplicación
@import url('01-tokens/tokens.css');

button {
  background: var(--color-action);
  color: var(--color-on-action);
}
```

### Para aplicaciones JavaScript
```javascript
// 1. Exportar tokens JS
python3 scripts/export-tokens.py --format js

// 2. Importar en tu app
import { semanticTokens, brandTokens } from './01-tokens/tokens.js';

const btnColor = semanticTokens.color_action;
const brandPalette = brandTokens.promptea.dark;
```

### Para equipos de diseño (Figma)
```bash
# Exportar tokens JSON
python3 scripts/export-tokens.py --format json

# Importar en Figma usando:
# - Figma Dev Mode + MCP
# - Token Studio plugin
# - Scripts personalizados
```

## 🔄 Flujo de CI (próxima versión)

```yaml
# .github/workflows/build.yml
on: [push, pull_request]

jobs:
  validate-and-export:
    runs-on: ubuntu-latest
    steps:
      # Linting de diseño
      - run: python3 scripts/validate.py
      
      # Exportar tokens
      - run: python3 scripts/export-tokens.py --format css
      - run: python3 scripts/export-tokens.py --format js
      - run: python3 scripts/export-tokens.py --format json
      
      # Commit y push si hay cambios
      - run: git add 01-tokens/*.{css,js,json}
      - run: git commit -m "chore: regenerate tokens" || true
      - run: git push
      
      # Tests visuales (opcional con Percy)
      - run: python3 scripts/vrt-tests.py
```
